    • Add

    Adding a task into TaskForce

    Format:

    Reminder: add TASKNAME [d/DESCRIPTION] [t/TAG]...
    Deadline: add TASKNAME [d/DESCRIPTION] [et/END_DATE] [t/TAG]...
    Event: add TASKNAME [d/DESCRIPTION] [st/START_DATE] [et/END_DATE] [t/TAG]...

    Example:

  • add cleaning House d/pocket money t/important
  • add concert d/justin biber st/130116 2100 et/130116 2200 t/important t/expensive
  • add report d/cs2105 et/130116 1300 t/important t/schoolwork

    • Find

    To Find a tasks in the TaskForce using a keyword

    A KEYWORD refer to a word contain in TASKNAME or a number that refer to DAYS or WEEKS

    Format : 'find METHOD KEYWORD'

    'd/' List all events a number of days after today `find d/ -1`
    'w/' List all events in a week, after current week `find w/ 0`
    `e/` List all events with word appearing in name `find e/ ceremony`
    'dl/' List all deadlines with word appearing in name `find dl/ homework`
    'r/' List all reminders with word appearing in name `find r/ shine`
    'a/' List all events with word appearing in name `find a/ shoes`

    > * The search is not case sensitive. e.g `hans` will match `Hans`

    > * The order of the keywords does not matter. e.g. `Hans Bo` will match `Bo Hans`

    > * Only the name is searched.

    > * Only full words will be matched e.g. `Han` will not match `Hans`

    > * Persons matching at least one keyword will be returned (i.e. `OR` search). e.g. `Hans` will match `Hans Bo`
